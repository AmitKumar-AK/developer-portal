# Data Exchange Framework 2.0

Synchronize data between Sitecore and 3rd party systems.

  <Alert variant='warning' mb={4}>
    <AlertIcon />
    Data Exchange Framework version 2.0 requires Sitecore Experience Platform version 9.0.
  </Alert>
